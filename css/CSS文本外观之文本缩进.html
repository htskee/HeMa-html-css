北京地铁（Beijing Subway），是服务于中国北京市的城市轨道交通系统，也是国际地铁联盟（CoMET）的成员，其首条线路于1971年1月15日正式开通运营 [1] ，使北京成为中国首个开通地铁的城市 [2] 。 截至2022年12月，北京地铁运营线路共有27条，运营里程797.3千米 [6] ，车站472座（其中换乘站78座） [3] [86] 。 截至2022年12月，北京地铁在建线路12条 [4] [151] 。到2025年，北京地铁将形成线网由30条运营，总长1177千米的轨道交通网络 [5] 。 2022年，北京地铁年客运量为22.62亿人次 [6] 。截至2022年12月，北京地铁单日最高客运量为1375.38万人次（2019年7月12日） [7] 。
